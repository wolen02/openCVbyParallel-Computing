# openCVbyParallel-Computing
병렬 구조를 통한 openCV 활용

/race-condition
:race condition 이론을 설명하기 위한 코드 모음

/boundedQueue
:boundedQueue 이론을 설명하기 위한 코드 모음

/drop
:drop-oldest방식과 drop-newest 방식을 설명하기 위한 코드 모음

/multprocessing
:GIL, multi processing, batch 단위 전달 방식에 대한 이론을 설명하기 위한 코드 모음

/openCV
: 실시간 영상을 받아 사람의 위치를 추론하여 보여주는 기능을 병렬 파이프라인 구조로 작성한 코드
, batch 방식을 사용한 코드와 사용하지 않은 코드 있음
